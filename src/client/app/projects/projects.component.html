Attaq Online Tournament Platform

A web app that automates the formation, progression, and payout of tournaments for eSports. Currently the game League of Legends is only supported.

I worked with a small team to create this Single Page Application using AngularJs 1.5 with a back-end service in nodeJs using the SenecaJs microservice framework. The app is intended to be used on desktop computers only as the primary users are using the app to participate in tournaments for games for desktop. The app consumes a RESTful API that is a wrapper for Riot Games' Tournament and Stats API.
